# openclawuse
Record the process of using OpenClaw.

## 常用命令

| 命令 | 功能 |
|------|------|
| `openclaw --version` | 查看OpenClaw当前版本信息 |
| `openclaw gateway status` | 查询网关服务的当前运行状态，包括服务是否正在运行、响应时间、连接数及关键组件健康状态等信息 |
| `openclaw status` | 查询openclaw主服务的整体运行状态，包括服务进程状态、版本信息、资源占用情况及核心功能模块的运行状态等关键指标 |
